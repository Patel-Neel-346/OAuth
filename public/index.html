How It Works

  1. Click one of the OAuth buttons above to begin the authentication flow
  2. After successful authentication, you'll be redirected back with an authorization token
  3. The token will be stored in cookies for future authenticated requests
  4. You can use the Swagger documentation to explore all available API endpoints

Test Your Authentication

Manual API Testing

You can also use the following curl commands to test the API:

# Register a new user
curl -X POST http://localhost:3000/auth/register \\
  -H "Content-Type: application/json" \\
  -d '{"name":"Test User","email":"test@example.com","password":"password123"}'

# Login with credentials
curl -X POST http://localhost:3000/auth/login \\
  -H "Content-Type: application/json" \\
  -d '{"email":"test@example.com","password":"password123"}'
